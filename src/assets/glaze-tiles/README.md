# Serie Taína — glaze test-tile photos

Drop the five test-tile photos into **this folder** with these exact
filenames (lowercase). They'll show up automatically on the Shop and in
the Gallery's "first glaze tests" section — no code changes needed.

| File          | Glaze     | Color        | The photo to use                          |
| ------------- | --------- | ------------ | ----------------------------------------- |
| `carey.jpg`   | Carey     | Turquoise    | the two turquoise tiles standing up       |
| `casabe.jpg`  | Casabe    | White        | the white / cream crackle tile            |
| `atabey.jpg`  | Atabey    | Cobalt Blue  | the vivid periwinkle-blue tile            |
| `conuco.jpg`  | Conuco    | Brown        | the tan / brown speckled tile             |
| `yuquiyu.jpg` | Yuquiyú   | Green        | the pale sage-green tile                  |

`.jpg`, `.jpeg`, `.png`, and `.webp` all work. Until a file is added,
that glaze shows a color placeholder in its hue.

Glaze definitions live in `src/data/glazes.js`.
